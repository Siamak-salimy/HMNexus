HMNexus

A high-speed, code-free GUI for downloading TCGA data

Explore the Website   Releases

Note: HMNexus is the official name of the tool formerly referred to as "TCGADownloader" in early development.

Abstract

Background: The Cancer Genome Atlas (TCGA) is a pivotal resource for cancer genomics, yet its practical use remains challenging for non-programmers due to reliance on command-line tools and scripting-based workflows. Although the NCI GDC Data Transfer Tool supports manifest-based downloads, it lacks a graphical interface and interactive data selection.

Results: We present HMNexus, an open-source, tkinter-based desktop application that enables intuitive graphical selection and high-speed, manifest-driven downloads from the GDC portal. Users can browse TCGA datasets by cancer type, data category, and sample attributes through a point-and-click interface. HMNexus employs optimized concurrent HTTP requests based on GDC manifest files, achieving up to 4.8× faster downloads compared to sequential command-line approaches, while requiring zero coding. Downloaded data are automatically organized into analysis-ready directory structures.

Key Features

  • Zero coding required — fully graphical interface
  • High-speed concurrent downloads via GDC manifests
  • Interactive filtering by cancer type and sample attributes
  • MD5 checksum validation for data integrity

Getting Started

Prerequisites: Python 3.9+ and pip.

  1. Download Releases (recommended) or run from source.
  2. From source: git clone https://github.com/Siamak-salimy/HMNexus.git → pip install -r requirements.txt → python main.py
